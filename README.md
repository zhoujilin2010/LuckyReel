# LuckyReel · 幸运摇奖机

老虎机式彩票选号前端 —— 拨动摇杆，逐球摇出真实随机号码。

## 特性

- 🎰 **老虎机式摇杆交互**：拨动摇杆，卷轴滚动定格 1 球，逐球摇完，模拟真实摇奖机
- 🔐 **密码学安全随机**：`crypto.getRandomValues` + 完整球池多轮 Fisher-Yates 洗牌 + **不放回**抽取（物理摇奖机模型）
- 🎯 **覆盖主流彩种**：双色球、大乐透、快乐8；支持单式 / 复式 / 多注（每注独立摇号）
- 📋 **选号单一键复制**：抬头写「彩种 · 玩法」，按第 1 注 / 第 2 注排列，含合计总价
- 📱 **移动优先**：单文件、零依赖、零构建，手机浏览器直接打开即可使用
- 🎨 **任天堂 Famicom 配色**：红 `#E60012` / 蓝 `#0078F0` / 黄 `#FBD000` / 奶白 `#F0F0E8`

## 用法

用手机或桌面浏览器直接打开 `index.html` 即可。建议「分享 → 添加到主屏幕」以便随时访问。

## 随机性说明

随机引擎移植自 [DestinyDraw](https://github.com/zhoujilin2010/DestinyDraw-) 项目：

> 安全随机源 → 构建完整球池 → 多轮洗牌混匀 → 逐个滚出（不放回）

**「自动摇完」与「手动拨杆」使用完全相同的抽球函数 `drawOne()`**，随机性完全一致；自动模式只是省去滚动动画，给出的是纯粹的密码学抽签结果。

## 支持彩种

| 彩种 | 球池 | 玩法 |
|---|---|---|
| 双色球 | 红 1–33 / 蓝 1–16 | 单式 6+1、复式 R 红 + B 蓝（R≥6） |
| 大乐透 | 前 1–35 / 后 1–12 | 单式 5+2、复式自定义 |
| 快乐8 | 单一池 1–80 | 选 1–选 10，单式 N 球 / 复式更多球 |
